# 医疗器械信息查询

- 国家药品监督管理局：[官方网站](http://www.nmpa.gov.cn/WS04/CL2042/)

- 查询方法：导航栏“医疗器械“ -> 左侧目录栏”医疗器械查询“ -> ”国产器械“ -> 右侧目录栏”医疗器械“ -> 查询相应产品包装上的编号，获得产品审批信息

- 说明：目前有些（新）审批的医疗器械的信息好像在国家药监局网站查不到，不知为何。
